# 🔄 워크플로우 개요

!!! quote "워크플로우 통합 자동화 요약"
    MDRM 워크플로우는 복잡한 IT 운영 절차를 시각적인 순서도로 설계하고 자동화하는 핵심 기능입니다.
    
    * **시각적 프로세스 설계**: 드래그 앤 드롭 방식으로 컴포넌트를 배치하여 복잡한 시나리오를 손쉽게 구성합니다.
    * **유연한 흐름 제어**: 조건 분기, 수동 확인, 승인 절차를 결합하여 기업의 업무 프로세스에 최적화된 흐름을 구현합니다.
    * **강력한 변수 시스템**: 글로벌/로컬/매크로 변수를 활용하여 동적인 데이터 전달과 지능적인 작업 수행이 가능합니다.
    * **실시간 관용 및 제어**: 실행 중인 작업의 실시간 로그 확인, 일시 중지, 재시도 및 성공 처리를 통해 예외 상황에 유연하게 대응합니다.
    * **체계적 결과 관리**: 상세한 실행 이력, 변수 정보, RTO 분석 결과를 포함한 실행 결과서를 제공합니다.

---

## 🖥️ 메인 화면 구성

워크플로우 메뉴에 진입하면 좌측 트리 메뉴와 연동된 통합 관리 화면이 나타납니다.

1. **트리 메뉴**: 데이터센터와 그룹, 워크플로우의 계층 구조를 관리합니다. 우클릭 컨텍스트 메뉴를 통해 추가/수정/삭제가 가능합니다.
2. **서브 메뉴**: 선택한 자원에 따라 **워크플로우 목록**, **스케줄 관리**, **글로벌 변수 정의** 화면을 제공합니다.
3. **목록 수 및 필터**: 페이지당 표시 개수 설정과 워크플로우 상태(잠금, 실행 여부, 성공/실패 등)에 따른 정밀 필터링을 지원합니다.
4. **일괄 작업 메뉴**: 여러 워크플로우를 선택하여 실행 취소/종료, RTO 일괄 변경, 담당자 변경, 일괄 이동 등을 수행합니다.
