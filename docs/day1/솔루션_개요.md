# 📘 솔루션 개요 및 요구사항

## **MDRM 솔루션 개요**

MDRM 솔루션은 마이크로서비스 아키텍처(MSA) 기반의 컨테이너 환경에서 유연하게 동작합니다.

### **기본 환경 사양**

| 구분 | 주요 내용 |
| :--- | :--- |
| <span style="white-space: nowrap;">**관리 콘솔**</span> | **Web 브라우저** (Google Chrome 최신 버전 권장) |
| <span style="white-space: nowrap;">**컨테이너 엔진**</span> | **Docker**, **Podman** |
| <span style="white-space: nowrap;">**운영체제(OS)**</span> | **Rocky Linux**, **RHEL** |
| <span style="white-space: nowrap;">**하드웨어(HW)**</span> | **VM** (Virtual Machine) 또는 **물리 머신** |

### **MDRM 솔루션 컨테이너 구성**

솔루션 설치 시 다음과 같은 주요 마이크로서비스 컨테이너들이 생성됩니다.

| 컨테이너명 | 설명 |
| :--- | :--- |
| <span style="white-space: nowrap;">`mdrm-ui`</span> | 워크플로우 설정을 위한 사용자 UI 서비스를 제공합니다. |
| <span style="white-space: nowrap;">`gam`</span> | 기존 MDRM의 핵심 기능을 포함한 통합 이미지입니다. |
| <span style="white-space: nowrap;">`dashboard-ui`</span> | 시스템 현황 모니터링을 위한 대시보드 UI를 제공합니다. |
| <span style="white-space: nowrap;">`mdrm-iam`</span> | 사용자 인증 및 권한 관리를 담당합니다. (Keycloak 기반) |
| <span style="white-space: nowrap;">`mdrm-base`</span> | 워크플로우 실행, 스케줄링, 알림 등 핵심 로직을 관리합니다. |
| <span style="white-space: nowrap;">`nginx-gateway`</span> | 서비스별 Rest API 요청에 대한 라우팅 처리를 담당합니다. |
| <span style="white-space: nowrap;">`mdrm-logstash`</span> | 실시간 로그 데이터를 수집하고 정제하여 처리합니다. |
| <span style="white-space: nowrap;">`mdrm-rabbitmq`</span> | 서비스 간 비동기 메시지 통신을 위한 메시지 큐입니다. |
| <span style="white-space: nowrap;">`mdrm-postgres`</span> | 시스템 운영 데이터를 저장하는 메인 데이터베이스입니다. |

## **하드웨어 요구사양**

MDRM 서버의 하드웨어 사양은 관리 대상 서버(Agent 설치 대수) 규모에 따라 결정됩니다. 다음 권장 사양 가이드를 참고하여 서버를 준비하십시오.

| 관리 대상 서버 수 | 권장 서버 사양 | 최대 동시 접속자 |
| :--- | :--- | :--- |
| <span style="white-space: nowrap;">**서버 50대 이하**</span> | • CPU: 2.0GHz 8Core 이상<br>• RAM: 16GB 이상<br>• Disk: 200GB 이상 | 20명 |
| <span style="white-space: nowrap;">**서버 100대 이하**</span> | • CPU: 2.0GHz 16Core 이상<br>• RAM: 24GB 이상<br>• Disk: 500GB 이상 | 50명 |
| <span style="white-space: nowrap;">**서버 500대 이하**</span> | • CPU: 2.0GHz 24Core 이상<br>• RAM: 32GB 이상<br>• Disk: 800GB 이상 | 100명 |

## **웹 브라우저 및 해상도 권장사항**

!!! tip "최적의 접속 환경"
    MDRM 관리 콘솔은 최신 웹 표준을 준수하며, **Google Chrome** 브라우저에서 가장 안정적으로 동작합니다.

    **지원 브라우저 안내:**
    - Google Chrome (가장 권장함)
    - Microsoft Edge
    - Firefox

    **권장 해상도 안내:**
    - 원활한 모니터링을 위해 **1920 x 1080** 이상의 해상도를 권장합니다.
