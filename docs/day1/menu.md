# 📚 MDRM 메뉴 설명

## 주요 메뉴 구성

| 메뉴 | 설명 |
|:---|:---|
| :material-view-dashboard: **대시보드** | 전체 시스템 현황 모니터링 |
| :material-server: **시스템** | 서버 및 시스템 관리 |
| :material-sitemap: **워크플로우** | 작업 흐름 설정 및 관리 |
| :material-clipboard-check: **점검작업** | 시스템 점검 및 작업 수행 |

---

## 메뉴별 상세 설명

### 대시보드

!!! info "대시보드"
    전체 시스템의 현황을 한눈에 볼 수 있는 모니터링 화면입니다.
    
    - 시스템 상태 요약
    - 최근 작업 현황
    - 알림 및 이벤트

### 시스템

!!! info "시스템"
    관리 대상 서버 및 시스템을 등록하고 관리합니다.
    
    - 서버 등록/삭제
    - 시스템 그룹 관리
    - Agent 상태 확인

### 워크플로우

!!! info "워크플로우"
    자동화된 작업 흐름을 설정하고 관리합니다.
    
    - 워크플로우 생성/편집
    - 실행 스케줄 설정
    - 실행 이력 조회

### 점검작업

!!! info "점검작업"
    시스템 점검 및 유지보수 작업을 수행합니다.
    
    - 점검 작업 생성
    - 수동 실행
    - 결과 확인
