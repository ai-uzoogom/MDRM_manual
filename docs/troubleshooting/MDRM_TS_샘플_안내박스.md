# 🧪 마크다운 안내 박스(Admonition) 샘플

!!! info "학습 안내"
    이 페이지는 MDRM 교육 매뉴얼 작성 시 사용할 수 있는 다양한 안내 박스(Admonition)의 종류와 스타일을 확인하기 위한 테스트 페이지입니다.

---

## **1. 기본 안내 박스 종류**

가장 자주 사용되는 표준 안내 박스들입니다.

!!! note "기본 참고 (Note)"
    일반적인 참고 사항을 전달할 때 사용합니다. 파란색 테두리와 연필 아이콘이 특징입니다.

!!! abstract "핵심 요약 (Abstract/Summary)"
    복잡한 내용을 요약할 때 사용합니다. 

!!! info "정보 제공 (Info)"
    중요한 정보를 전달할 때 사용합니다. 현재 우리 매뉴얼의 '학습 안내'에 사용 중인 스타일입니다.

!!! tip "꿀팁/권장사항 (Tip)"
    최적의 방법이나 노하우를 공유할 때 사용합니다. 전구 아이콘이 표시됩니다.

!!! success "성공/완료 (Success)"
    작업이 성공적으로 끝났거나 권장되는 모범 사례를 보여줄 때 사용합니다.

!!! question "도움말/질문 (Question)"
    궁금한 점이나 FAQ, 도움말 섹션에 적합한 녹색 아이콘 박스입니다.

---

## **2. 주의 및 경고 박스**

사용자의 주의가 필요한 상황에서 사용하는 강조 블록입니다.

!!! warning "주의 사항 (Warning)"
    실수하기 쉬운 부분이나 가벼운 주의가 필요할 때 사용합니다. 주황색으로 표시됩니다.

!!! failure "실패/오류 (Failure)"
    작업 실패나 발생 가능한 오류 상황을 설명할 때 사용합니다.

!!! danger "심각한 위험 (Danger)"
    데이터 유실이나 시스템 중단 등 치명적인 위험이 있을 때 사용하는 빨간색 박스입니다.

!!! bug "버그 리포트 (Bug)"
    알려진 버그나 기술적 결함에 대해 기술할 때 사용합니다.

---

## **3. 기타 및 고급 스타일**

!!! example "사용 예시 (Example)"
    코드 스니펫이나 구체적인 실행 사례를 보여줄 때 유용합니다.

!!! quote "인용구 (Quote)"
    참고 문헌이나 다른 가이드의 인용이 필요할 때 사용합니다.

---

## **4. 접기/펼치기 효과 (Collapsible)**

내용이 길어 페이지를 많이 차지할 때 사용자가 클릭하여 열어볼 수 있도록 만듭니다.

??? info "상세 내용 보기 (클릭하면 열립니다)"
    `!!!` 대신 `???`를 사용하면 아코디언 스타일의 접기 메뉴가 생성됩니다.

???+ tip "기본으로 펼쳐진 박스 (???+ 사용)"
    뒤에 `+`를 붙이면 페이지 로드 시 기본적으로 내용이 펼쳐진 상태로 노출됩니다.
