# 📊 대시보드

!!! info "학습 안내"
    MDRM의 실시간 모니터링 시스템의 핵심인 대시보드의 주요 기능과 구성 요소, 그리고 운영 목적에 맞는 대시보드 구축 방법을 학습합니다.


MDRM 콘솔에 로그인하면 가장 먼저 대시보드 화면이 나타납니다. 관리 자원들을 실시간으로 모니터링할 수 있는 대시보드와 이를 구성하는 다양한 위젯들에 대해 학습합니다.
