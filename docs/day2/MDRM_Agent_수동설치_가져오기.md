# 📥 Agent 가져오기 (Import)

!!! info "학습 안내"
    수동으로 설치된 Agent를 MDRM 서버의 관리 자원으로 명시적으로 등록하는 [시스템 가져오기] 절차를 학습합니다.


---

## **1. 시스템 가져오기 기능 개요**

수동으로 설치된 Agent는 MDRM 서버와 통신은 가능하나, 관리 목록에는 자동으로 나타나지 않습니다. 따라서 **[시스템 가져오기]** 기능을 통해 명시적으로 등록해주어야 합니다.

*   **위치**: 시스템 > 시스템 등록 및 추가 > **시스템 가져오기**
*   **작동 원리**: MDRM 서버가 입력된 IP와 포트로 Agent에 접속하여 인증 정보를 확인하고 시스템 메타데이터를 가져옵니다.

---

## **2. 가져오기 절차**

1.  **MDRM 웹 콘솔**에 로그인합니다.
2.  좌측 메뉴에서 **시스템** 메뉴로 이동합니다.
3.  상단 버튼 중 **[시스템 가져오기]**를 클릭합니다. (시스템 추가 버튼 옆의 드롭다운 또는 별도 버튼 확인)
4.  팝업창에서 다음 정보를 입력합니다:
    *   **접속 이름**: 대상 서버의 이름을 입력합니다.
    *   **접속 정보**: 대상 서버의 IP 주소를 입력합니다.
5.  **[사전 점검]**을 클릭하여 Agent와 정상적으로 통신이 되는지 확인합니다.
6.  점검 결과가 '성공'이면 **[가져오기]** 버튼을 눌러 등록을 완료합니다.

---

## **3. 완료 확인**

*   가져오기가 완료되면 트리 메뉴의 지정된 그룹 하위에 해당 시스템이 나타납니다.
*   **상태** 아이콘이 파란색(정상)으로 표시되는지 확인하십시오.
