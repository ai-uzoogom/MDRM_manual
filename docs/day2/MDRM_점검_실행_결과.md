## 1. 작업 이력 확인
작업별 '작업 이력' 탭에서 과거부터 최신까지의 모든 실행 기록을 추적할 수 있습니다.

### 1.1 주요 집계 정보
- **작업 시간**: 시작, 종료, 전체 소요 시간 기록
- **실행 방식**: 사용자가 직접 실행한 `수동(Manual)`인지 스케줄에 의한 `자동(Auto)`인지 구분
- **서버/항목 수**: 해당 이력 시점의 점검 규모 확인
- **최종 준수율**: (준수 수 * 100) / (전체 서버 수 * 전체 항목 수)

---

## 2. 상세 결과 분석
단순 수치를 넘어 서버별, 항목별 원본 메시지를 확인하여 장애 원인을 파악합니다.

### 2.1 서버별/항목별 전환 뷰
- **서버별 상세**: 특정 서버에서 어떤 항목들이 실패했는지 확인
- **항목별 상세**: 특정 점검 항목(예: CPU 임계치)이 어떤 서버들에서 미준수 상태인지 일괄 확인

### 2.2 미준수 건 조치 관리
- **조치/계획 설정**: '미준수' 상태인 항목에 대해 조치 완료 여부나 향후 계획을 입력하여 관리 피드백을 남깁니다.
- **메시지 확인**: 실행 결과 메시지를 클릭하면 스크립트가 반환한 전체 Raw 데이터를 팝업으로 확인할 수 있습니다.

---

## 3. 결과 리포트 (Excel)
점검 결과를 보고서 형식으로 추출하여 외부 보고용으로 활용할 수 있습니다.

- **리포트 타입**: `서버별 상세결과` 또는 `항목별 상세결과` 중 선택 가능
- **엑셀 다운로드**: 필터링된 현재 화면의 정보를 그대로 엑셀 시트로 변환하여 저장 및 인쇄할 수 있습니다.
