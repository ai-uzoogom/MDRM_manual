# 📊 관리 및 실행 리포트

워크플로우의 변경 이력을 관리하고, 실행 결과를 정량적으로 분석하여 리포트로 추출합니다.

---

## 1. 워크플로우 운영 관리

### 변경 이력 및 복원 (Version Control)
*   추가/수정된 모든 내역을 버전별로 저장합니다. 
*   **복원 기능**: 특정 시점의 [이 버전으로 복원] 버튼을 클릭해 설계를 과거 상태로 되돌릴 수 있습니다.

### 잠금 설정 (Locking)
최고관리자 권한으로 워크플로우의 오작동 및 임의 수정을 방지합니다.
*   **편집 잠금**: 복제, 삭제, 편집이 불가능하나 실행은 허용.
*   **실행 잠금**: 편집과 실행 모두 차단하고 정보 확인만 가능.

---

## 2. 실행 이력 및 결과 데이터

실행이 종료된 워크플로우의 상세 데이터를 추적합니다.

*   **실행 내역**: 컴포넌트별 시작/종료 시간, 상태, 실제 경과 시간 확인.
*   **변수 정보**: 실행 당시 각 변수에 저장되었던 실제 결과값을 스냅샷으로 제공.
*   **실행 결과서 (Excel)**: 
    *   워크플로우 단계를 지정(1단위~전체)하여 엑셀 파일로 다운로드.
    *   링크 컴포넌트 내부 결과 포함 여부 선택 가능.

---

## 3. RTO (목표 복구 시간) 관리

시스템 복구 목표와 실제 수행 시간을 시각적으로 비교 및 관리합니다.

*   **RTO 차트**: 컴포넌트별 목표 시간(RTO)과 실제 소요 시간을 그래프로 대비하여 병목 구간 파악.
*   **RTO 설정**: 
    *   워크플로우 전체 및 개별 컴포넌트의 목표 시간 정의.
    *   [과거 실행 이력 가져오기]를 통해 최근 수행 시간을 RTO 기준으로 자동 설정.
*   **RTO 일괄 변경**: 선택한 워크플로우들의 모든 컴포넌트 RTO를 최신 실행 이력 기반으로 일합 업데이트.

---

## 4. 워크플로우 절차서

설계된 프로세스를 문서화된 매뉴얼 형태로 추출합니다.

*   **목적**: 비숙련자도 절차를 파악할 수 있도록 텍스트 기반의 가이드 제공.
*   **기능**: 단계별(Depth) 출력 범위 조절, 실행 명령 포함 여부 선택 후 엑셀 다운로드 지원.
