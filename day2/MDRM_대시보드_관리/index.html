    📑 대시보드 관리¶

    대시보드는 관리 자원들의 상태 정보를 한눈에 모니터링할 수 있는 메인 화면입니다. 사용자는 자신의 목적에 맞게 대시보드를 추가하고 관리할 수 있습니다.

    1. 대시보드 메인 화면 (UI)¶

    대시보드 모니터링 화면은 다음 6가지 주요 영역으로 구성되어 있습니다.

    1. 대시보드 선택: 클릭 시 현재 생성되어 있는 대시보드 목록이 표시됩니다. 최초 지원 시 'MDRM 소개' 대시보드가 기본으로 제공됩니다.
    2. 대시보드 관리: 버튼을 클릭하여 위젯을 추가하거나 관리합니다. '더보기' 버튼을 통해 공유, 기본 설정, 삭제 기능을 사용할 수 있습니다.
    3. 대시보드 공유: 자신이 관리하는 대시보드를 다른 사용자나 그룹에 공유할 수 있습니다. 공유 시 '수정 권한'을 선택하여 부여할 수 있습니다.
    4. 전체 화면 전환: 탑 메뉴를 숨기고 대시보드 영역을 화면 전체로 확장합니다.
    5. 위젯 영역: 대시보드를 구성하는 개별 모니터링 프로그램 영역입니다. 5초 주기로 실시간 갱신됩니다.
    6. 다크 모드 지원: 화면 우측 하단의 테마 변경 버튼을 통해 다크 모드와 라이트 모드를 전환할 수 있습니다. 설정된 테마는 사용자별로 저장됩니다.

    2. 대시보드 추가 및 편집¶

    대시보드 추가 절차¶

    1. 대시보드 선택 창을 클릭 후, 하단의 [대시보드 추가] 버튼을 클릭합니다.
    2. 좌측 상단에 대시보드 이름을 입력합니다.
    3. 우측의 [위젯 추가] 탭에서 원하는 위젯을 배치한 후 저장합니다.

    대시보드 편집¶

    메인 화면 우측의 버튼을 클릭하면 편집 화면으로 이동합니다. 위젯의 추가, 삭제, 배치를 자유롭게 변경할 수 있습니다.

    3. 대시보드 공유 및 관리¶

    대시보드 공유¶

    • 공유 그룹/사용자 선택: MDRM에 등록된 전체 사용자 및 그룹 목록 중 대상을 선택합니다.
    • 수정 권한 부여: 수정 권한을 부여받은 사용자는 이름 변경, 위젯 변경, 공유 해제가 가능합니다.

    상세 관리 기능¶

    대시보드 선택 창 우측의 버튼을 클릭하여 관리 팝업창에서 다음 설정을 수행합니다. * 순서 변경: 목록 앞의 아이콘을 드래그하여 순서를 조정합니다. * 기본 대시보드 지정: 로그인 후 가장 먼저 보여지는 대시보드를 설정합니다. 기본 대시보드는 삭제할 수 없습니다. * 숨기기: 목록에서 일시적으로 숨김 처리합니다. * 삭제: 불필요한 대시보드를 영구 삭제합니다.
