# 📂 구성 및 그룹 관리

!!! info "학습 안내"
    데이터센터와 그룹을 활용하여 관리 자원을 체계적으로 조직화하고, 실시간 상태 아이콘을 통해 시스템 가동 현황을 직관적으로 파악하는 방법을 학습합니다.


---

## **1. 데이터센터 메인 화면 구성**

메인 화면은 실시간 관제와 자원 탐색을 위해 크게 5가지 영역으로 구분됩니다.

1. **트리 메뉴**: 데이터센터-그룹-시스템을 계층 구조로 표시합니다. 우클릭을 통해 관리 메뉴(추가/수정/삭제 등)를 사용합니다.
2. **서브 메뉴**: 선택한 자원에 따라 시스템 상태, 계정, 변경 요약, 알림 정보를 제공합니다.
3. **시스템 검색 바**: 시스템 이름이나 IP 주소로 신속하게 자원을 검색합니다. (설치 중인 시스템 제외)
4. **시스템 뷰 전환**: 직관적인 아이콘 중심의 **그리드 뷰**와 상세 정보 중심의 **테이블 뷰**를 지원합니다.
5. **실시간 상태 표시**: 5초 주기로 갱신되는 아이콘 색상을 통해 시스템의 생존 여부와 장애 상태를 즉각 파악합니다.

---

## **2. 데이터센터 및 그룹 관리**

트리 메뉴를 활용해 데이터센터별, 업무 그룹별로 시스템을 조직화합니다.

### 데이터센터 관리
* **추가**: 트리 공백 마우스 우클릭 → [데이터센터 추가]. 시스템, 워크플로우, 점검 메뉴와 실시간으로 동기화됩니다.
* **삭제**: 내부에 그룹이나 시스템이 존재할 경우 삭제가 제한되므로, 먼저 하위 항목을 비워야 합니다.

### 그룹 및 시스템 관리
* **그룹화**: 데이터센터 내에 하위 그룹을 생성하여 관리 단위를 세분화합니다.
* **정렬/이동**: 드래그 앤 드롭으로 시스템 위치를 자유롭게 이동할 수 있으며, 오름차순/내림차순 정렬 기능을 제공합니다.
* **이름 변경**: 우클릭 메뉴를 통해 데이터센터나 그룹의 명칭을 영구적으로 변경합니다.

---

## **3. 시스템 상태 식별 가이드**

아이콘의 모양과 색상은 시스템의 현재 상태를 의미합니다.
